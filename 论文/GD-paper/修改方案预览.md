```tex
针对家庭储能、分布式光伏和离网备用供电对高效率、高可靠性和智能化管理的需求，
本文设计了一套多MCU协同光储一体化数字电源系统控制平台。
系统采用非隔离三端口功率变换架构，将光伏端口、储能端口和直流母线端口统一到

多级安全保护和无线状态监控。功率级硬件已完成仿真验证与板级设计，为后续闭环
控制提供了基础。初步设计表明，该方案具备结构紧凑、控制灵活、扩展性好等特点，
```

```tex
\subsection{三端口功率变换硬件设计}

本系统主功率部分采用面向光伏输入、电池储能和负载输出的三端口功率变换方案。
该方案的设计目标是在统一功率级中组织光伏端、储能端和输出端之间的能量流动
关系，为后续实现光伏输入利用、电池充放电控制和输出稳压提供硬件基础。

主功率电路主要包括PV输入端、BAT储能端、VOUT输出端、四绕组耦合电感、
主功率开关器件、续流与钳位器件以及采样调理电路。PV端用于接入光伏输入，
BAT端用于连接储能电池，VOUT端用于形成后级直流输出。四绕组耦合电感作为
能量传递的核心磁性器件，在不同端口之间建立磁耦合关系。

功率器件选用IRFP4668PBF、IPB073N15N5、BSC0702LS和GC4D05120A等，
由主控制板输出的\texttt{PWM\_M1}、\texttt{PWM\_MA}、\texttt{PWM\_M2}和
\texttt{PWM\_M3}信号控制。驱动部分采用UCC21540ADWKR隔离驱动芯片，
为各开关支路配置独立隔离电源，并预留\texttt{OC\_FAULT}故障接口与主控
HRTIMER故障输入配合。

采样信号包括光伏输入电压$V_{\mathrm{PV}}$、输出电压$V_{\mathrm{OUT}}$、
光伏输入电流$I_{\mathrm{PV}}$、输出电流$I_{\mathrm{OUT}}$、MOSFET温度
$TEMP_{\mathrm{MOS}}$和磁性器件温度$TEMP_{\mathrm{MAG}}$。电压采样采用
高阻分压与OPA2188缓冲；电流采样采用分流电阻配合INA181检测芯片；温度
采样用于监测功率器件热状态。

目前，三端口功率板已完成仿真验证、原理图设计和PCB板级实现。板级设计中
主功率区域、驱动区域、采样区域和接口区域进行了功能分区。实物测试将按照
由低压到高压、由开环到闭环、由单端口到多端口协同的顺序推进。
```

```tex
\subsection{BMS产品状态机}

GD32C113 BMS工程采用产品状态机管理运行流程，主要包含三种状态：

\textbf{（1）INIT\_FAIL状态}

系统上电后进入初始化阶段，依次完成：
\begin{enumerate}[label=\arabic*),leftmargin=*,itemsep=0.3ex]
  \item 板级初始化（时钟、GPIO、通信接口）
  \item AFE最小唤醒（GD30BM2016寄存器配置）
  \item FET通路关闭、均衡清零
  \item 完整AFE配置：16S配置、CCGain设置、保护项装载
\end{enumerate}

若初始化过程中AFE通信失败、配置异常或自检不通过，系统进入INIT\_FAIL状态，
保持FET关闭并上报初始化故障。

\textbf{（2）RUN状态}

当首次测量有效、无严重故障且电芯状态正常时，系统进入RUN状态。每个控制周期
依次执行：
\begin{enumerate}[label=\arabic*),leftmargin=*,itemsep=0.3ex]
  \item 测量模块：读取AFE电压、电流、温度数据
  \item 保护模块：过压、欠压、过流、过温判断
  \item SOC模块：OCV估算与库仑计累计
  \item 均衡模块：判断单体电压差并执行被动均衡
  \item FET模块：根据保护状态和充放电允许条件更新FET策略
  \item 通信模块：组织CAN FD遥测帧并发送
\end{enumerate}

% TODO: 插入BMS状态机状态转移图
\begin{figure}[htbp]
  \centering
  \fbox{\parbox[c][3.5cm][c]{0.88\linewidth}{\centering
  此处插入BMS状态机图：INIT → RUN → DEGRADED
  }}
  \caption{BMS产品状态机}
  \label{fig:bms_state_machine}

\textbf{（3）DEGRADED状态}

若运行过程中出现以下任一情况，系统立即进入DEGRADED状态：
\begin{itemize}[leftmargin=*,itemsep=0.3ex]
  \item AFE读取失败或通信超时
  \item 保护项触发（过压/欠压/过流/过温）
  \item FET应用失败
  \item 严重安全异常
\end{itemize}

进入DEGRADED后，系统执行均衡抑制和FET全关断，停止充放电通路，
待故障恢复后重新评估是否返回RUN状态。
```
