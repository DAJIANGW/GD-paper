\section{方案论证与设计}

\subsection{总体方案}

系统采用“前级光储三端口变换 + 后级单相逆变 + 多 MCU 协同控制”的总体架构。前级三端口变换器负责光伏输入、储能电池充放电和直流母线稳压；后级逆变器将直流母线能量转换为 220 V/50 Hz 单相交流输出；BMS 模块对电池状态进行采样、估算、均衡和保护；无线监控模块负责参数配置、运行状态显示和故障告警。

系统的核心设计思想是将强实时任务与弱实时任务分离。功率变换闭环、PWM 更新和快速保护由 GD32G553 就近完成；电池状态估算、均衡和电池侧保护由 GD32C113 独立完成；无线通信、数据记录和人机交互由 GD32VW553 承担。通过任务分层，系统避免了无线通信和日志处理对功率控制中断的干扰。

\subsection{方案选择与论证}

在光储逆变一体化系统中，常见方案包括分立式多级变换方案和集成式多端口变换方案。分立式方案结构清晰、调试难度较低，但光伏、储能和逆变模块之间需要多次能量变换，控制接口复杂，整机体积和成本较高。集成式三端口方案能够在同一功率级内协调光伏和电池端口，减少中间环节，有利于提高功率密度和系统动态响应，但其控制耦合更强，对数字控制器和保护策略提出更高要求。

本作品选择非隔离三端口功率变换架构，主要基于以下考虑：

\begin{enumerate}[label=\arabic*、, leftmargin=*, itemsep=0.8ex, parsep=0pt, topsep=0.5ex]
  \item 家庭小型离网备用供电场景更关注体积、成本和效率，非隔离方案在满足安全设计前提下具有较高集成度。
  \item 光伏端和电池端电压等级接近，适合通过多端口变换器实现能量双向或准双向协调，减少独立充电器和独立升压级。
  \item GD32G553 具备较丰富的定时器、ADC 和运算资源，可承担多路采样、PWM 输出和闭环控制任务，适合实现数字电源算法。
  \item 独立 BMS 控制器可以降低主功率控制器的软件复杂度，使电池安全保护不依赖单一 MCU。
\end{enumerate}

\subsection{控制芯片分工}

\subsubsection{GD32G553 与主功率控制}

GD32G553 作为系统主控，负责前级三端口变换器和后级逆变器的核心控制。其主要任务包括：采集光伏电压电流、电池端口电流、直流母线电压、逆变输出电压电流；运行 MPPT、母线电压环、电流环和逆变电压控制算法；生成三端口变换器和逆变桥 PWM；处理过流、过压等快速故障中断。

\subsubsection{GD32C113 与 BMS}

GD32C113 负责电池侧安全管理。其采样对象包括单体电压、总压、充放电电流和关键温度点。软件上实现 SOC 估算、均衡控制、充放电允许状态判断和故障码生成。当检测到过压、欠压、过温或通信异常时，BMS 通过通信接口向主控上报状态，并可触发充放电限流或关断策略。

\subsubsection{GD32VW553 与无线监控}

GD32VW553 负责无线通信和人机交互。其主要功能包括接入局域网、上传电压电流功率数据、记录故障日志、接收参数配置命令和提供运行状态显示接口。无线模块不直接参与高速功率闭环，只通过主控约定的参数接口更新限值和工作模式，避免网络延迟影响功率控制稳定性。

\subsection{主功率电路仿真方案}

主功率电路仿真分为开环验证和闭环验证两个阶段。开环阶段使用 LTspice 或等效电路仿真工具验证三端口拓扑的电压增益、电感电流波形、器件电压应力和开关节点波形；闭环阶段使用 Simulink 或等效控制仿真环境验证 MPPT、母线稳压、负载阶跃和模式切换过程。
